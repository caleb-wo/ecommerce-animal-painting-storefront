# Animal Painting Ecommerce App

This project was generated with Angular CLI version 17.3.17. It was a part of a Udemy course I enrolled in to learn the basics of Angular. This project simulates a simple ecommerce site which sells animal paintings. I'm currently working on it right now.

## Angular Tooling Learned & Used

- Angular v17
- RxJs Observables
- Angular Material
- Angular Routing
- Angulr HttpClient
- 2-way data binding
